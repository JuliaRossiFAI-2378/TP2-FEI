Patterns Portfolio is a clean, modern block-based WordPress theme built for designers, photographers, artists, and creative professionals. It provides a flexible, full-site-editing experience with intuitive portfolio layouts that let you beautifully showcase your work and projects.

== Description ==

Patterns Portfolio is a stylish and versatile Full Site Editing (FSE) WordPress theme, ideal for creatives looking to showcase their work online. Whether you're a designer, photographer, artist, or freelancer, this theme helps you build a professional digital portfolio with ease.

Built with WordPress Full Site Editing (FSE), it offers complete control over headers, footers, templates, and global styles — no coding needed. The theme includes pre-designed patterns and layouts for project showcases, case studies, galleries, testimonials, about sections, and contact forms.

With its responsive design and elegant presentation, Patterns Portfolio provides a modern platform to highlight your skills, attract clients, and grow your creative presence.
